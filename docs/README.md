## 当前后端版本能力

当前后端已经覆盖用户端主流程：

```text
分类 -> 商品 -> 登录 -> 地址 -> 购物车 -> 下单 -> 模拟支付 -> 确认收货 / 取消订单
```

当前已补充工程化能力：

- 统一响应结构
- 全局异常处理
- DTO 请求对象
- 参数校验
- JWT 认证
- 用户所有权校验
- 业务异常和系统异常分离
- 接口测试

## 阅读顺序

建议按以下顺序阅读：

1. 先看需求分析，了解项目边界。
2. 再看接口文档，了解前后端如何对接。
3. 再看 JWT 认证设计，了解 token 和 `/current` 接口。
4. 最后看后端完善说明，了解异常、校验和后续优化。
